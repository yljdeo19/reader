Проект представляет собой стильное веб-приложение для чтения книг и визуальных новелл.
Главная страница содержит геро-блок, подборки книг, а также динамические карточки, которые загружаются из MongoDB.
У каждой книги есть собственная страница и возможностью перехода к чтению.

Проект выполнен в мягкой аниме-палитре (soft lavender) на Jakarta EE, с акцентом на эстетичный UI.
